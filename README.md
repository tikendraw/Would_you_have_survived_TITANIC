# Would_you_have_survived_TITANIC

This App predicts weather you could have survived the TITANIC CRASH  or not.

### DATASET has been:
* Balanced 
* Scaled
* Cleaned
* Nan Filled
* Encoded

### Model : XGBCLassifier 
Out of Many Classification Models XGBClassifier was the best Performing.

### Installtion/Run
```
# make it excutable
chmod 700 run.sh

# run it
bash run.sh

```
